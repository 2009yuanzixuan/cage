使用方法

在网页后面加上'/ststic/文件类型'

例如(/ststic/images,/ststic/music,/ststic/video)

然后加上'/文件名称'

例如[/123.png,/hhh.mp3]

然后使用我的功能函数爬取图片

             功能函数

                    def payin(url,clas,fn):

                         import requests

                         response = requests.get(url)

                         file_name=fn+clas

                         with open(file_name,"wb") as file:

                             file.write(response.content)

                         return file_name

解释

clas是类型

例如('.mp3','.mp4','.png')

fn是文件名

例如['123','hhh']

url是使用方法中的网址
